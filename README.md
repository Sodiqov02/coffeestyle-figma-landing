# CoffeeStyle Landing Page

Static Figma-to-code landing page reconstruction for a CoffeeStyle ecommerce concept.

## Stack

- HTML5
- CSS3
- Responsive CSS media queries
- Static assets exported from Figma

## Features

- Desktop layout rebuilt from the 1440px Figma frame
- Responsive product grids, promo block, newsletter, and footer
- Accessible form label, focus states, skip link, and descriptive image alt text
- Lightweight hover states for links, buttons, product cards, and story cards
- Static-host friendly relative asset paths

## Responsive Checks

Checked target widths:

- 320px
- 360px
- 390px
- 430px
- 768px
- 1024px
- 1440px

## Figma-To-Code Note

The implementation preserves the original CoffeeStyle visual direction, typography, spacing system, product-card structure, and desktop section order. Images were exported from the source Figma file and stored locally in `assets/images`.

## Deployment

This project is a static site and can be deployed to GitHub Pages, Netlify, Vercel, or any static host.

For local preview, open `coffee_style_landing_html.html` directly in a browser or serve the folder with any static file server.
